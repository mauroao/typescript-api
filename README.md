# typescript-api
Typescript API Project

## Prerequisites

- Visual Studio Code;
- Plugin: EditorConfig for VS Code;
- Node.js v16.3.0;

## Install dependencies

```
npm install
```

## Build

```
npm run build
```

## Lint

```
npm run lint
```

## Test

```
npm test
```
